# Megastore Busca Rust

## 📚 Descrição
Este projeto tem como objetivo simular um sistema de busca de produtos utilizando a linguagem Rust, aplicando estruturas de dados eficientes como HashMap.

## 🎯 Objetivo
Desenvolver um sistema simples de busca que permita armazenar e consultar produtos por nome.

## 🧠 Estrutura Utilizada
Foi utilizado o HashMap, uma estrutura de dados que permite armazenar pares chave-valor, garantindo buscas rápidas e eficientes.

## ⚙️ Funcionamento
- Produtos são inseridos no sistema com nome e marca
- O sistema permite buscar produtos pelo nome
- Caso o produto não exista, o sistema informa

## 💻 Tecnologias
- Rust
- HashMap (std::collections)

## ▶️ Como Executar
1. Instale o Rust na máquina
2. Execute o código com:
cargo run

## 📊 Exemplo de Uso
Busca por produto existente:
Produto encontrado: notebook - Marca: Dell

Busca por produto inexistente:
Produto tablet não encontrado

## 📈 Conclusão
O uso do HashMap demonstrou ser eficiente para sistemas de busca simples, permitindo acesso rápido aos dados.
